## Overview

CloudVault AI is a full-stack application featuring a React-based frontend providing a file upload interface (dropzone) and an AWS Serverless (SAM) Node.js API backend responsible for producing pre-signed URLs for direct-to-S3 file uploads. Data persistence revolves around DynamoDB for tracking file metadata.

## Data Flow

1. User visits frontend and drops a file onto `UploadDropzone.jsx`.
2. Frontend requests an upload URL from the configured AWS API Gateway endpoint (`/files/upload-url`).
3. API Gateway forwards the request to the `getUploadUrlFunction` Lambda.
4. Lambda generates a pre-signed URL via S3 permissions and optionally records metadata into DynamoDB `CloudVaultFiles`.
5. Frontend receives URL and directly uploads file to Amazon S3.
